Come “esportare” questo su Svelte/JS
1. Stili globali (Tailwind-like o CSS custom)

Se vuoi emulare il loro approccio, puoi:

Usare Tailwind e definire in tailwind.config.js:

js
theme: {
  extend: {
    colors: {
      dark: '#05050a',   // ipotetico bg
      brand: '#247DFF',  // theme-color
    },
    fontFamily: {
      poppins: ['Poppins', 'system-ui', 'sans-serif'],
    },
  },
}
In app.css / global.css Svelte:

css
body {
  @apply bg-dark text-white font-poppins antialiased;
}
Se non vuoi Tailwind, definisci classi equivalenti:

css
.bg-dark { background-color: #05050a; }
.text-white { color: #ffffff; }
.font-poppins { font-family: 'Poppins', system-ui, sans-serif; }
.antialiased { -webkit-font-smoothing: antialiased; -moz-osx-font-smoothing: grayscale; }
2. Font preload

Emula il pattern nel <head> di SvelteKit o template HTML:

xml
<link
  rel="preload"
  href="/fonts/poppins-regular.woff2"
  as="font"
  type="font/woff2"
  crossorigin
>
3. Tracking (se vuoi replicarlo)

TikTok Pixel in Svelte:

xml
<script>
  (function(w,d,t,u,s,q,m){
    w.ttq = w.ttq || [];
    q = w.ttq;
    q.push(['config', 'CLDION3C77U93A3DFSOG']);
    var js = d.createElement(t); js.async = true;
    js.src = 'https://analytics.tiktok.com/i18n/pixel/events.js?sdkid=CLDION3C77U93A3DFSOG&lib=ttq';
    var f = d.getElementsByTagName(t)[0];
    f.parentNode.insertBefore(js, f);
  })(window, document, 'script');
</script>
Google Tag / GTM copia-incolla con i tuoi ID (GA4, Ads, GTM) seguendo il pattern che vedi nei loro script.
